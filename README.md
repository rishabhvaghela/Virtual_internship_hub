# Virtual_internship_hub
This the website which is create an bridge between student and company.

## Features 

### Students 
1. register and login
2. Create Update profile
3. Apply for Online Internship
4. View Application Status
5. Build Resume
6. Can Give SKill Based Test & Improve Skills

### Company
1. Post Internship
2. View Application
3. Edit & Delete Internship
